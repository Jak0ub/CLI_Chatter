# Instalation

### Download
```
git clone https://github.com/Jak0ub/CLI_Chatter
cd CLI_Chatter
```
----
### Venv creation

#### `Unix based`

```
python3 -m venv venv
source venv/bin/activate
pip install -r req.txt
```
#### `Windows`


```
python -m venv venv
.\venv\Scripts\activate
pip install -r req.txt
```
